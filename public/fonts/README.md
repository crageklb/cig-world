# Web fonts

Add these font files for the CigWorld title:

1. **56th Street** (for "Cig")
   - File: `56th-street.woff2`, `56th-street.woff`, or `56th-street.otf`
   - Free from: https://jameslafuente.com/56th-street-free-font

2. **Druk Text Wide** (for "World")
   - File: `druk-text-wide.woff2`, `druk-text-wide.woff`, or `druk-text-wide.otf`
   - Commercial font from Commercial Type (or trial from CDNFonts)

The CSS loads woff2 first, then woff, then otf. Convert OTF to woff2 for best performance if needed.
